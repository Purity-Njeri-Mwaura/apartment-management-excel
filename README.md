# Apartment Management & Financial Tracking System (Excel)

## 📌 Project Overview
This project is an Excel-based Apartment Management System designed to track rental income, expenses, tenant information, and overall financial performance of a residential apartment.

The system helps property owners or managers monitor rent collection, expenses, arrears, deposits, and profitability using structured data and dashboards.

---

## 🏢 Key Features
- Rent tracking per housing unit
- Tenant and deposit management
- Expense and maintenance tracking
- Profit & Loss (P&L) summary by month
- Automated yearly totals
- Key performance indicators (KPIs) such as occupancy and collection rates
- Interactive dashboard for quick insights
